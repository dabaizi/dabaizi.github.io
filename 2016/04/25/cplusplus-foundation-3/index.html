【C++基础】静态成员

Apr 25, 2016

静态成员在C++中，以static关键字作为声明关键字，声明为静态成员的类成员函数或者成员变量能够在类的范围内共享，静态成员属于整个类，而并不是某个对象，该类的所有对象均可以共享静态成员。相对于普通数据成员，有几种不同：

1、普通类成员属于特定的某个对象，只有对象被创建了，普通数据成员才会被分配空间；而静态成员属于整个类，即使该类没有被创建对象，静态成员也存在。

2、由于静态成员的存在不依赖于某个对象，因此静态成员必须在代码中显示地初始化。一般情况下，在类的外部初始化

3、静态成员可以通过类名直接访问，也可以通过类的对象、指针或者引用来访问，但是普通成员就不能通过类名直接访问了

4、静态成员函数不能访问普通成员变量或者调用普通成员方法，反之则可行。

通过几个例子来详细说明：

实例1：
class A
{
public:
void f()
{
}
static void g()
{
}
};

int main()
{
A::f();
A::g();
system(“pause”);
return 0;
};

error C2352: “A::f”: 非静态成员函数的非法调用

非静态成员函数不能通过类名来调用，静态成员函数可以通过类名直接调用。
